VMware WorkStation配置Ubuntu Server网络

发表于 2016-10-13   |   分类于 Linux

VMware WorkStation之前的配置因为电脑经常连接不同的网络导致ip切换，使得虚拟机不能上网。

配置IP为固定，则可以不因网络变动导致虚拟机不能上网。这里采用NAT模式设置虚拟机网络。

步骤

  1. 查看VMWareNAT设置，记下网关IP

  2. 配置windows本地虚拟网卡VMnet8的IP为固定

  3. 配置静态IP

    root@master:~# vim /etc/network/interfaces

  4. 配置DNS

    查看本地DNS

    root@master:~# vim /etc/resolv.conf

    防止重启后overwrite配置
    root@master:~# vim /etc/resolvconf/resolv.conf.d/base

  5. 重新启动网络配置
    root@master:~# /etc/init.d/networking restart

  6. 测试是否成功

    ping www.qq.com
