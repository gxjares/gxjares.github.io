log4j使用

发表于 2016-09-22   |   分类于 java

日志是应用软件中不可缺少的部分，Apache的开源项目log4j是一个功能强大的日志组件，提供方便的日志记录。在apache网站可以免费下载到Log4j最新版本的软件包。

项目中使用log4j

配置文件

在java项目中src同级目录（JAVAWEB项目为应用程序的/WEB-INF/classes目录中，由Web容器自动调用）加入配置文件log4j.properties

java代码中使用日志

Appender

Log4j提供的appender有以下几种：

  1. RollingFileAppender
    这种日志默认以追加的方式写日志，一般需要设置日志文件大小和日志个数，否则日志会在默认日志文件写满之后继续生成一个新文件做备份，若继续写文件超过该容量的话，日志内容只会保存容量范围内最新的内容，会覆盖之前的日志文件。如果日志不多的话，可以使用这种方式，否则建议使用DailyRollingFileAppender，

  2. DailyRollingFileAppender
    DailyRollingFileAppender按天生成日志文件，每天一个文件，不会有大小限制。
